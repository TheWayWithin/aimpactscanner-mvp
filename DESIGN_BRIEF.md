# AImpactScanner Visual Design Brief
## For Professional Designer - MVP Image Requirements

### Project Context
AImpactScanner is a SaaS tool that analyzes websites for AI search engine visibility. The brand emphasizes trust, authority, and technical expertise while maintaining approachability. We need professional visual assets for our MVP launch.

### Brand Guidelines
- **Primary Colors**: 
  - Mastery Blue: #1E3A8A (primary brand elements)
  - Innovation Teal: #0891B2 (interactive elements, accents)
  - Authority White: #FFFFFF (backgrounds)
  - Framework Black: #0F172A (text)
  - AI Silver: #64748B (supporting elements)
  
- **Typography**: Inter, Source Sans Pro (already implemented)
- **Style Direction**: Professional, trustworthy, tech-forward but approachable
- **Target Audience**: SMB owners, marketers, SEO professionals

## TECHNICAL REQUIREMENTS

### Performance Optimization:
- All raster images must be provided in WebP format with PNG fallback
- Use appropriate compression (85% quality for JPEGs, optimize PNGs)
- SVGs should be optimized (clean code, no unnecessary metadata)
- Total image payload for initial page load should not exceed 500KB

### Accessibility:
- Provide suggested alt text for each image
- Ensure sufficient contrast for any text overlays
- Icons should be distinguishable in grayscale

### File Naming Convention:
```
logo-primary.svg
logo-dark-bg.svg
hero-illustration-desktop.webp
icon-ai-analysis.svg
tier-coffee.svg
badge-ssl-secured.svg
empty-state-no-results.svg
```
